## 📂 Project Setup

Follow these simple steps to customize and run the project:

### 1. Edit Configuration

Open the `setup.js` file using **Notepad** or any text editor of your choice.

### 2. Update Your Information

* Replace the text values inside the quotation marks (`" "`).
* **Do not remove the quotation marks.**
* Keep all dates in the **same format** as provided.

### 3. Save Changes

Save the `setup.js` file after making your edits.

### 4. Launch the Project

Open `index.html` in any web browser you like.
